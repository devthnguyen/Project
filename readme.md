
# My Projects

## 🛠️ Web Build PC
- Dự án web giúp người dùng lựa chọn linh kiện máy tính, được xây dựng bằng React, Node.js và MongoDB. 
- Ngoài việc phát triển chức năng, tôi tập trung vào kiểm thử API bằng Postman, kiểm tra giao diện người dùng và debug lỗi hệ thống nhằm đảm bảo dữ liệu và chức năng hoạt động chính xác.
- Link GitHub: https://github.com/devthnguyen/Project/tree/master/Web_BuildPc
