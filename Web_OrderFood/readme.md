# 🍔 Web Đặt Đồ Ăn

## 📌 Giới thiệu

Dự án web cho phép người dùng xem và lựa chọn món ăn, được xây dựng nhằm mục đích học tập và thực hành phát triển web.
Ngoài ra, dự án tập trung vào kiểm thử API và xử lý lỗi trong quá trình phát triển.

---

## 🚀 Công nghệ sử dụng

### 🔹 Frontend

* ReactJS (đang chuyển sang Angular)
* HTML, CSS, JavaScript

### 🔹 Backend

* Node.js
* Express.js

### 🔹 Database

* MongoDB

---

## ⚙️ Chức năng chính

* Hiển thị danh sách món ăn
* Thêm / sửa / xóa dữ liệu (CRUD)
* Gọi API từ backend
* Hiển thị dữ liệu từ database

---

## 🧪 Testing

* Kiểm thử API bằng Postman (GET, POST)
* Kiểm tra giao diện người dùng (UI)
* Debug lỗi dữ liệu giữa frontend và backend

---

## 📂 Cài đặt và chạy dự án

### 1. Clone repository

```bash id="g3s1az"

```

```bash id="zq0j2o"

```

### 3. Cài đặt dependencies

```bash id="r1j0cl"

```

### 4. Chạy dự án

```bash id="p9gk8w"

```
